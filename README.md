# TrueMachine — система оценки кандидатов

Сервис для автоматической сверки требований заказчика с навыками кандидатов по резюме.  
Позволяет менеджерам быстро оценивать соответствие кандидатов, получать процент покрытия и экспортировать отчёты.

---

## 📦 Требования

- [Laragon](https://laragon.org/download.html) (с PHP 8.3+)
- PostgreSQL 18 (устанавливается через Quick Add в Laragon)
- Node.js 20+ (устанавливается через Laragon)
- Composer (устанавливается через Laragon)
